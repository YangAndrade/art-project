🎨 Art Project – Galeria em Realidade Virtual (Unity + Meta SDK)
📌 Sobre o Projeto

O Art Project é uma aplicação simples desenvolvida na Unity utilizando o Meta SDK, com o objetivo de criar uma pequena galeria de arte em Realidade Virtual (VR).

O projeto apresenta um ambiente 3D com cenário ambientado e quadros distribuídos como uma exposição virtual, permitindo ao usuário explorar o espaço de forma imersiva.
--------------------------------------------------
🛠 Tecnologias Utilizadas

🎮 Engine: Unity

🥽 SDK de VR: Meta XR SDK

🧰 Plataforma alvo: Dispositivos Meta Quest
---------------------------------------------
🖼 Funcionalidades

Ambiente 3D estilizado simulando uma galeria de arte

Quadros posicionados nas paredes do cenário

Navegação em realidade virtual

Interação básica com o ambiente (movimentação e exploração)
------------------------------------------------------------
🚀 Como Executar o Projeto
1️⃣ Pré-requisitos

Unity (versão compatível com o Meta SDK)

Meta XR SDK instalado

Meta Quest (ou dispositivo compatível)

Meta Quest Link ou Air Link configurado

2️⃣ Passos

Clone ou baixe este repositório.

Abra o projeto na Unity.

Verifique se o Meta XR SDK está configurado corretamente.

Conecte o dispositivo Meta Quest ao computador.

Altere a plataforma para Android em: File > Build Settings > Android > Switch Platform

🎯 Objetivo do Projeto

Este projeto foi criado com finalidade educacional e experimental, com foco em:

Aprender integração de VR com Unity

Utilizar o Meta SDK

Criar ambientes imersivos simples

Organizar elementos visuais em um espaço 3D
